Anaconda Navigator一直卡在加载页面，我来帮你解决

发表于2025-07-19|更新于2025-07-19
|总字数:471|阅读时长:1分钟

前言

最近要学习数据分析与可视化，需要用到anaconda，我记得电脑上的anaconda还是三四年前想学习机器学习然后安装的，所以版本非常老旧，当我跟着视频教程走的时候，叫我打开anaconda图形界面？？？我迷茫了

我的anaconda老到图形界面都没有，所以我选择重新安装anaconda，版本是2025.06，具体怎么查看它的版本号我不太懂，我在官网下载的，地址为：Download Success | Anaconda，我选择的Distribution ，然后下载时，下载的文件上会写。

当我重新安装anaconda却遇到了一系列问题分别是：

 1. 一直卡在那个加载页面，雷打不动
 2. VS Code自己打开，不关闭anaconda就卡死，我会弹出两次，要分别关闭了才能继续成功打开

记录一下我的解决办法

卡在init或者loading界面或者其他

我看了好多博客，但是几乎都是很早很早的版本了，但是我了解到，当我们启动anaconda时，它会加载一些国外的东西，所以非常非常慢，那我们怎么做呢？让它超时。

首先进入anaconda目录下的Lib\site-packages\anaconda_navigator\utils\attribution

打开resources.py文件修改第21行代码，找不到可以CTRL+F查找一下timeout

这就是让我们苦苦等待的原因，我们将它的超时时间设置为0.1，问题得到解决。

VS Code自己打开

首先我们先打开anaconda图形界面，选择左上角的File>Preferences

打开后，直接划到最底下，找到VS Code Path

然后将你电脑上VS Code的路径填入，例如我这里安装在E盘E:\Microsoft VS Code下，那么直接将这个路径填入即可，然后Apply，现在就可以顺利打开了，下课！！！
