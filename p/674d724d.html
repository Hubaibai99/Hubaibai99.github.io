一、文本格式快捷键

功能 Windows/Linux Mac Markdown 语法 补充说明
加粗 Ctrl + B ⌘ + B 加粗内容 选中文本后使用，直接为文本添加加粗格式
斜体 Ctrl + I ⌘ + I 斜体内容 选中文本后使用，支持嵌套（如 粗斜体）
下划线 Ctrl + U ⌘ + U 下划线内容 部分 Markdown 解析器需开启扩展支持
删除线 Alt + Shift + 5 ⌘ + Shift + 5 删除线内容 选中文本后使用，常用于标记作废内容
行内代码 Ctrl + Shift + `` ⌘ + Shift + `` 行内代码 选中文本后使用，自动包裹代码，避免语法冲突
插入超链接 Ctrl + K ⌘ + K 链接文本 可直接粘贴 URL，支持本地文件路径（./xxx.md）
插入图片 Ctrl + Shift + I ⌘ + Shift + I 支持拖拽图片自动生成路径，alt 文本用于加载失败提示
自动补全符号 Tab Tab - 输入 (/[/

二、标题与段落快捷键

功能 Windows/Linux Mac Markdown 语法 补充说明
一级标题 Ctrl + 1 ⌘ + 1 # 一级标题 光标在段落内即可使用，无需全选
二级标题 Ctrl + 2 ⌘ + 2 ## 二级标题 按快捷键后可直接输入标题内容
三级标题 Ctrl + 3 ⌘ + 3 ### 三级标题 -
四级标题 Ctrl + 4 ⌘ + 4 #### 四级标题 -
五级标题 Ctrl + 5 ⌘ + 5 ##### 五级标题 -
六级标题 Ctrl + 6 ⌘ + 6 ###### 六级标题 最低级标题，常用于小节标注
转为正文 Ctrl + 0 ⌘ + 0 普通文本 取消标题格式，恢复为正文段落
块引用 Ctrl + Shift + Q ⌘ + Shift + Q > 引用内容 多次使用可嵌套（如 >> 二级引用）
段落左对齐 Ctrl + Shift + L ⌘ + Shift + L - 默认对齐方式，适用于大部分文本
段落居中 Ctrl + Shift + C ⌘ + Shift + C - 常用于标题、图片等元素的居中展示
段落右对齐 Ctrl + Shift + R ⌘ + Shift + R - 较少用，适用于特殊格式需求（如落款）
段落两端对齐 Ctrl + Shift + J ⌘ + Shift + J - 适用于长文本段落，优化排版美观度
水平分割线 Ctrl + Shift + - ⌘ + Shift + - — 或 *** 需单独占一行，避免与上下文本粘连

三、列表与任务快捷键

功能 Windows/Linux Mac Markdown 语法 补充说明
无序列表 Ctrl + Shift + L ⌘ + Shift + L - 列表项 或 * 列表项 按快捷键后输入内容，回车自动生成下一项
有序列表 Ctrl + Shift + O ⌘ + Shift + O 1. 列表项 自动递增序号，删除中间项后序号会自动重排
任务列表（待办） 手动触发 手动触发 - [ ] 待办任务 输入 - [ ] （注意空格）后按空格触发，无需快捷键
任务列表（完成） 手动触发 手动触发 - [x] 完成任务 将 [ ] 改为 [x]（x 不区分大小写）
列表层级缩进 Tab Tab - 增加列表嵌套层级（如子列表）
减少列表缩进 Shift + Tab Shift + Tab - 减少嵌套层级，回到上一级列表

四、代码与数学公式快捷键

功能 Windows/Linux Mac Markdown 语法 补充说明
行内代码 Ctrl + Shift + `` ⌘ + Shift + `` 代码片段 适用于单行代码（如 print(“hello”)）
代码块 Ctrl + Shift + K ⌘ + Option + C 语言 代码块 生成后可在开头指定语言（如 python）开启语法高亮
行内数学公式 Ctrl + Shift + M ⌘ + Option + M $ 公式内容 $ 适用于短公式（如 $a^2 + b^2 = c^2$）
数学公式块 Ctrl + Shift + M ⌘ + Option + B $$ 公式内容 $$ 单独占一行，支持复杂公式（如矩阵、积分）

五、表格操作快捷键

功能 Windows/Linux Mac Markdown 语法 补充说明
插入表格 Ctrl + T ⌘ + T ` 列 1
表格添加行 Alt + Shift + ↓ ⌥ + Shift + ↓ - 光标在目标行任意位置，按快捷键添加下方行
表格删除行 Alt + Shift + ↑ ⌥ + Shift + ↑ - 光标在目标行，按快捷键删除当前行
表格移动行 Alt + ↑ / Alt + ↓ ⌥ + ↑ / ⌥ + ↓ - 上下移动当前行，保持表格结构不变
表格合并单元格 手动操作 手动操作 - 选中多个单元格，右键选择 “合并单元格”

六、视图与模式快捷键

功能 Windows/Linux Mac 补充说明
专注模式 F8 F8 仅高亮当前段落，隐藏其他内容，适合专注写作
打字机模式 F9 F9 光标始终保持在屏幕中间，减少视线移动
源代码模式 Ctrl + / ⌘ + / 切换为纯 Markdown 语法视图，可手动修改代码
大纲面板 Ctrl + Shift + 1 ⌘ + Shift + 1 显示文档标题结构，支持点击跳转
全屏模式 F11 ⌃ + ⌘ + F 隐藏窗口边框，最大化编辑区域
预览模式 Ctrl + Shift + P ⌘ + Shift + P 部分版本支持，仅展示渲染后的效果（非实时）

七、文档与窗口快捷键

功能 Windows/Linux Mac 补充说明
新建文档 Ctrl + N ⌘ + N 新建空白 Markdown 文档
打开文档 Ctrl + O ⌘ + O 支持批量选择多个文档同时打开
保存文档 Ctrl + S ⌘ + S 自动保存为 .md 格式，建议定期保存
另存为 Ctrl + Shift + S ⌘ + Shift + S 可修改文件名、保存路径或格式（如 PDF）
打开最近文件 Ctrl + Shift + O ⌘ + Shift + O 显示最近编辑的文档列表，快速恢复
关闭文档 Ctrl + W ⌘ + W 未保存时会提示是否保存
退出 Typora Alt + F4 ⌘ + Q 关闭所有文档并退出程序

八、查找与跳转快捷键

功能 Windows/Linux Mac 补充说明
文本查找 Ctrl + F ⌘ + F 支持大小写敏感、全字匹配，按 Enter 跳转下一个
文本替换 Ctrl + H ⌘ + Option + F 可批量替换文本，支持预览替换结果
跳转文档开头 Ctrl + Home ⌘ + ↑ 快速回到文档第一行
跳转文档结尾 Ctrl + End ⌘ + ↓ 快速定位到文档最后一行
跳转指定标题 大纲面板点击 大纲面板点击 在大纲面板中点击标题，直接跳转至对应位置
